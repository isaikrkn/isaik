# Isaik site — v3

This version improves spacing and margins, keeps English as default, adds EN/ES/FR language switching, and repositions the site toward cybersecurity, automation, cyber-physical systems, and sonic research.

## Local preview

```bash
python3 -m http.server 8000
```

Open:

```text
http://localhost:8000
```

## GitHub Pages

Upload all files to your repository and enable Pages from:

- **Settings → Pages**
- **Source:** Deploy from a branch
- **Branch:** `main`
- **Folder:** `/ (root)`

## Notes

- Update the LinkedIn URL if your public profile slug changes.
- Replace email, GitHub, and external links if needed.
- The language choice is stored in `localStorage`.
